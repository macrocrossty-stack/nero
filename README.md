# 眠り姫と黒豹の物語（仮）

ティラノスクリプト製ノベルゲームの制作リポジトリ。
企画の原本は Notion「🌳眠り姫と黒豹の物語」。決定事項は `docs/` に反映して管理する。

## いますぐ読む（プレビュー）

**`preview/index.html` をブラウザで開くだけ**で、プロローグ〜第2章を通し読みできます。
（開発用の簡易プレビューです。クリック / Enter で読み進め、右上のボタンで章ジャンプ）

シナリオ（.ks）を書き換えたあとは、スナップショットを再生成してください:

```
python3 tools/build_preview.py
```

## ティラノスクリプトへの組み込み方（本実装）

1. https://tyrano.jp/ から ティラノスクリプト V5 をダウンロード
2. エンジン一式を展開（`index.html` と `data/` フォルダがあるもの）
3. このリポジトリの `game/data/` の中身を、エンジンの `data/` に上書きコピー
   - `data/scenario/first.ks` などが置き換わります
4. エンジンの `index.html` をティラノスタジオ等で起動

※ BGM/SE はまだ未配置です。シナリオ内の `[playbgm]` はコメントアウトしてあるので、
素材を `data/bgm/` に置いてからコメントを外してください。

## フォルダ構成

```
docs/                       制作ドキュメント
  01_企画書.md               作品概要・章構成・伏線管理
  02_キャラクター設定.md      少女＆ネロの設定と「書き方ガイド」
  03_素材リスト.md            背景/スチル/BGM/SE の必要素材一覧
  04_UI画面構成.md            UI設計・演出ルール
  05_寓話_黒猫と姫の物語.md    劇中絵本（ドラフト版・要差し替え確認）
game/data/                  ティラノスクリプトの data/ に対応
  scenario/                 シナリオ本体 (.ks)
    first.ks                エントリーポイント（簡易タイトル）
    prologue.ks             プロローグ「絵本」
    chapter1.ks             第1章「邂逅」
    chapter2.ks             第2章「力の芽生え」
  bgimage/                  背景画像（現在はすべて自動生成の仮素材）
preview/                    ブラウザ用簡易プレビュー
tools/
  make_placeholder_bg.py    仮背景ジェネレータ（要 pillow）
  build_preview.py          プレビュー用スナップショット生成
```
