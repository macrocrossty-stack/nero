# assets — ゲーム内画像の置き場

ここに画像ファイルを置いて、シナリオの `img:` 行にパスを足すと表示されます。

```
img: 犬の写真（フレブル・ブリンドル） | assets/e04_guri.png
```

ファイルが無い間は自動でダミー枠になるので、**シナリオに先にパスを書いておいてもOK**です（描けたものから順に置くだけで反映される）。

## 推奨仕様

| 用途 | 推奨サイズ | 備考 |
| --- | --- | --- |
| チャット内の写真（縦横比 自由） | 幅 160px 前後のドット絵 | 表示幅220pxに `image-rendering: pixelated` でくっきり拡大される。倍サイズ（320px）でも可 |
| 陽菜のアイコン | 32×32 | ヘッダー表示用（🌙と🐶の差し替え、2種） |
| フォーマット | PNG（透過可） | ファイル名は `e04_guri.png` のように話数_内容で |

## 命名規則

`e{話数}_{内容}.png` — 例: `e04_guri.png`, `e06_nail.png`, `e08_shelf.png`
