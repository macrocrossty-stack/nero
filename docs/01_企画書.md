## 2. 物語の設計（決定版）

### プレイヤー体験の設計
- **序盤**: 秘密を握る「不思議な存在」は黒豹だと思わせる。ヒロインは等身大の女の子＝プレイヤーの分身として信頼させる。
- **中盤**: 記憶の欠落の発覚（力の芽生えとセット。恐怖として書く）。
- **後半**: ヒロインが機械的に"豹変"することで、「自分が操作していた」と思っていたプレイヤーの足場を崩す。エンジン演出（選択肢の乗っ取り等）を併用。
- **導入のミスリード**: 絵本前編（猫が王子に戻る話）を先に読ませ、「傷ついた黒豹を人間に戻す物語」だと思い込ませる。

### 世界観の理屈（設定の背骨・決定版）
**敵は「忘却」、少女は「世界の記憶を歌い直す装置」、燃料は「彼女自身の記憶」。**

- **空の災厄** = 世界に「自分の形を忘れさせる」宇宙的侵食。太陽が消える・色が濁る・名前が失われる。絵本後編の「お日さまが消えた」と直結。
- **影の怪物** = 忘れられたものたちの成れの果て。見る者の恐怖を映すのは、「思い出されよう」ともがくから。恐れるほどに形を得て育つ。
- **少女** = 古代人が世界の記憶と歌を注ぎこんで作った器（調律装置）。調律のたび、**自分の記憶を燃やして**世界の形を歌い直す。記憶喪失は副作用ではなく**代金**。
- **感情は記憶を重く、明るくする** → 幸せになるほど調律は強力になり、失うものも増える。ネロはそれを知りながら、彼女に幸せな日常を与えている（本作の感情的エンジン）。
- **名前は記憶の錨であり、調律の鍵**。絆で結ばれた者が真の名を呼ぶことが最終解放のトリガー。ネロが名前を呼ばないのは作法ではなく、「呼ぶこと＝終わらせること」だから。
- **ネロ** = 元は古代の研究員/警備兵。少女を庇って瀕死になり、彼女の力で「黒豹」として固定された。毎夜、森の外周で影を狩り、侵食を遅らせている（＝彼女の平穏なひとり暮らしは、彼が血を流して維持している舞台装置）。

## 3. 演出・伏線（本編に織り込む約束事）

1. **名前呼び**: ネロは「お嬢様」と呼ぶ（第2章で、名乗れない彼女への配慮として発生）。終章で一度だけ名前を呼ぶ＝最終解放のトリガー。
2. **絵本**: プロローグで前編朗読。第1章で「昔から破れている」最終ページ。第3章で後編発見。クリア後に修復。
3. **研究日誌**: 少女には読めず、プレイヤーだけが読める。「七度目の記録」。
4. **音の設計**: 災厄側＝ガラスの軋む音 / 少女の力＝ガラスの共鳴音（清濁で対比。毎回同じ音）。
5. **違和感の種**: カップ2組・止まった大時計・埃のない部屋・毎朝増えるネロの傷（すべて後章で回収）。
