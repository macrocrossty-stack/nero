# 企画書 — 『眠り姫と黒豹の物語』（仮）

## 1. 作品概要

| 項目 | 内容 |
|---|---|
| ジャンル | ファンタジー・ロマンス・SF（叙事詩的） |
| 形式 | ノベルゲーム（ティラノスクリプト） |
| 画面 | 横画面 16:9（基準解像度 1280×720） |
| 想定プレイ時間 | 2〜4時間（全5章構成・現在は第2章まで執筆） |
| エンド | ループエンド（一本道。選択肢は没入感を壊さない範囲で最小限） |

### トーン & リファレンス
- **背景**: 劇場版エスカフローネ的なリアル描写 ＋ レジェンドオブマナ的な遊び心
- **空気感**: 酒井駒子、『ウルフウォーカー』——「陰影のある幻想」
- **文体**: 一人称（少女視点）を基本に、地の文は静かで感覚的。プロローグ/ラストの寓話パートのみ「語り」の文体

### コアテーマ
1. 星を超える愛
2. 避けられない別れと再会（ループ）
3. 黒豹の健気な献身

## 2. あらすじと構成

- **導入のミスリード**: 「傷ついた黒豹を人間の姿に戻す物語」だと思わせて始まる。
- **真実**: 少女こそが「宇宙的災厄を抑える兵器/器」。黒豹は彼女を守るために人であることを捨てた従者。
- **敵**: 特定の魔王ではなく「空の災厄（宇宙的侵食）」。見る者の恐怖を反映して「影の怪物」として現れる。
- **結末**: 少女は世界を守るために力を解放し、記憶を失って再び眠りにつく。黒豹はそれを覚えており、次の目覚めを待つ。

### 章構成（案）

| 章 | タイトル | 内容 | 状態 |
|---|---|---|---|
| プロローグ | 絵本 | 寓話「黒猫と姫の物語」の朗読 | ✅ 脚本あり |
| 第1章 | 邂逅 | 目覚め・黒豹との出会い・「呪いを解く」というミスリードの成立 | ✅ 脚本あり |
| 第2章 | 力の芽生え | 影の怪物の初襲撃・黒豹の変身・少女の力の兆し | ✅ 脚本あり |
| 第3章 | 屋敷の秘密 | 研究日誌の発見・プレイヤーだけが真実に近づく | 未着手 |
| 第4章 | 空の災厄 | 侵食の本格化・禁忌形態の解放・真実の告白 | 未着手 |
| 終章 | 眠り姫 | 力の解放・名前呼び・ループエンド・寓話の再演 | 未着手 |

## 3. 演出・伏線（本編に織り込む約束事）

1. **名前呼び**: 黒豹は少女を一貫して「お嬢様」と呼ぶ。少女がねだっても断る。眠りにつく寸前に一度だけ名前で呼ぶ。
   - → 第1章に「名前をねだって断られる」シーンを配置済み。
2. **絵本の寓話**: 冒頭で「黒猫と姫の物語」が語られ、ラストで自分たちの物語だったと判明する。
   - → プロローグで朗読。第1章の書庫に同じ絵本を置き、「最後のページが破れている」ことを提示済み。
3. **屋敷の資料**: 「古代の研究日誌」が隠されており、プレイヤーだけが記憶リセットの不可避性を察する。
   - → 第2章ラストで日誌の断片を初出し（少女には読めない文字＝プレイヤーだけが読める演出）。
