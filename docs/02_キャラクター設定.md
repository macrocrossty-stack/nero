# キャラクター設定（決定版）

## 少女（主人公）／真の名は仮・ステラ（終章まで一切非表示）

| 項目 | 内容 |
|---|---|
| 外見 | 10代半ば。白いワンピース。淡い色の髪（スチルで確定） |
| 暮らし | 森の屋敷でひとり暮らし。料理も掃除も自分でこなす |
| 人物像 | **童話の中の子**。礼儀と常識はあるが、世界への驚き方が少しだけ本のなかの人みたい（＝正体の伏線） |
| 一人称 | わたし |
| 口調 | やわらかい。「〜ですね」「〜かなあ」。ヨルには最初から敬語を使わない |
| 記憶 | 喪失の**自覚がない**。彼女の世界は欠けて見えず、完結して見える（夢の中で不条理に気づかないのと同じ。疑問を持つ足場の記憶ごと無い）。**発覚は終盤の調律発動と同時**——燃やされる直前の記憶が一瞬すべて灯り、思い出すのと理解するのが同時に来る |
| 正体 | 世界に自分を思い出させる「歌い手」（器・調律装置）。書庫は彼女の食事（世界の記憶の保管庫）。本の虫なのはそのため |
| 情緒 | 実質稼働時間が短く、最初は子供っぽい「不思議系」。旅と解呪クエストを通じて育つ。**恋の自覚は眠りに落ちる寸前、ただ一度**（「そっか。わたし、あなたのこと——」で言葉は完成しない） |
| 立ち絵 | 基本なし。"豹変"（調律状態）・重要シーンのみ |

### 書き方のガイド
- 第1章の彼女は何も欠けていない普通の女の子。孤独を寂しがらないことの異常さは、地の文の明るさで隠す。
- 「名前が思い出せない」等の明示ギミックは**禁止**。思い出話をしない（エピソード記憶の不在）は、初見で気づけないレベルに沈める。
- 知識はあるのに経験がないちぐはぐさ（パンの焼き方は手が覚えているのに、はじめて焼く）。
- 中盤以降、自分の力を「呪いを解く力」と**自分で誤読**する。この間違った仮説が彼女の行動原理。
- 恐怖の対象は自分の正体ではなく「ヨルが傷つくこと」「この日常が終わること」。
- "豹変"では一人称の地の文に等幅・無機質な文体が混入する（エンジン演出と連動）。
- 真実を知ったときの反応は、怒りでも取り乱しでもなく——「ずっとひとりで持ってたんだね、これ」。

## 黒豹（従者）／ヨル

| 項目 | 内容 |
|---|---|
| 名前 | **ヨル**。今周期に彼女が命名（「夜のあいだだけ人に戻れるから」）。人間だった頃の名は変容の代金として世界から失われ、本人も思い出せない。過去周期の名（クロ、ノワール、ヨイ、ネロ…）は旅先の言い伝えに残る |
| 外見（人間時） | 20代半ば。執事服。黒髪、金の瞳（黒豹と同じ瞳＝伏線） |
| 外見（黒豹時） | 大型の黒豹。左肩に深い傷（第1章で彼女が手当てする。人間時も同じ位置） |
| 登場 | 第1章で傷ついた黒豹として拾われる。第2章で「昔このお屋敷にお仕えしていた者」と名乗る（→お嬢様呼びの自然発生＝偽の来歴をプレイヤーに自作させる） |
| 表向きの設定（彼の創作） | 呪いで獣にされた元使用人。夜のわずかな間だけ人に戻れる。名前も呪いに喰われた。**記憶喪失のフリ**（ただし名前の喪失だけは真実） |
| 正体 | 元は古代の研究員/警備兵。器が最初に認識した人間（刷り込み）として、システムに「**種火**」指定された。周期ごとに白紙になる彼女に、世界の愛し方をもう一度教える役目。死ぬことも離れることも許されない——**これが本物の呪い** |
| なぜ黒豹か | 瀕死の彼を、生まれたての器が調律で繋ぎ止めた。彼女の中にあった温かい形は「黒い猫のお話」ただひとつ。「このひとは死なない。だって黒い猫だから」——彼を守れる大きさの黒い猫＝黒豹。**彼は彼女の最古の、燃えない記憶でできている**（毎周期「なつかしい」と感じる理由） |
| 夜だけ人に戻れる理屈 | 夜は世界が夢を見る時間。世界の記憶が緩み、夢の中でだけ彼はまだ人間として思い出される（災厄＝忘却と同じ原理の、優しい側の顔） |
| 傷の理由 | 毎夜、森の外周（旅中は野営の外）で影を狩り、侵食を遅らせている。毎朝新しい傷が増える |
| 性格 | 寡黙、忠誠、自己犠牲。感情は瞳と行動にだけ出る |
| 口調 | 敬語。「〜でございます」「なりません」。短文。戦闘時のみ一人称が「俺」に揺れる |
| 戦闘形態 | ①人間・剣 → ②黒豹 → ③禁忌。**必ずこの順で戦う理由（決定設定）**: 剣は人間だった頃の彼自身の技術＝「彼が彼であるため」の戦い方。豹の体は彼女の最古の記憶が編んだもので、これで戦うたび世界が彼を「獣」として記憶していき、夜に人へ戻れる時間がすり減る（変身のたび、彼女と話せる夜を支払っている）。禁忌は人と豹の記憶が混ざり「どちらでもなくなっていく」姿＝使うほど戻れなくなる。第5章の台詞「この体は、あなたにいただいたものですから。これで血を流すのは——惜しいのです」で一度だけ明示 |
| 絵本 | 『姫と黒猫の物語』の作者。毎周期、書庫に置く。後編を破り取ったのも彼。2周目ラストで、七周期で初めて**続きを書き始める** |

### 書き方のガイド
- 嘘はつかないが、真実を言わない。解呪クエストは彼の創作だが、「呪いを解く鍵は、あなたなのかもしれません」のように**すべての層で真実になる台詞**を選ばせる。
- **解呪クエストは推進する**（彼女の日々が輝くほど装填が進む／旅は遺跡への誘導）。**恋愛の距離だけは全力で避ける**（未練ごと歌わせないため）。この非対称が中盤の違和感＝伏線になる。
- 抑圧は所作で漏らす: 躱した直後に尻尾が一度だけ床を打つ／命名の瞬間に一拍言葉を失う／絵本の綴じ目を直す慣れた手つき。
- 献身も所作で: 毒見、扉の前の気配、竈の火加減。
- ユーモアは無自覚に。真顔でずれたことを言う（帽子パン）。
- 彼の「お嬢様」は距離ではなく祈り。名前を呼ばないのは作法ではなく「呼ぶこと＝終わらせること」だから。

## 敵：空の災厄（忘却）

| 項目 | 内容 |
|---|---|
| 正体 | 宇宙そのものの忘却。世界が自分の形を忘れていく。太陽が消える・色が濁る・名前が失われる |
| 影の怪物 | 忘れられたものの成れの果て。「思い出されよう」ともがくため見る者の恐怖を映し、恐れるほど形を得て育つ |
| 被害の点描 | 顔だけ褪せる肖像画／名前を失くした橋／「空」という名前を忘れていく空（終盤） |
| 音 | 災厄側＝ガラスの**軋む**音 ／ 彼女の力＝ガラスの**共鳴**音（清濁の対比、毎回同一SE） |

## 関係性の推移

| 章 | 少女→ヨル | ヨル→少女 |
|---|---|---|
| 1章 | 拾った「絵本みたいな」黒豹 | （豹として）沈黙の庇護 |
| 2章 | 夜に現れる不思議な大人。質問攻め | 七周期め の命名を受け取る |
| 3章 | 解呪クエストの相棒。「なんか、へん」の芽 | 想定外の後編発見に動揺。距離の防波堤を守る |
| 4章 | 名前のつかない、あたたかくて少し苦しいもの | 抑圧の限界。所作に漏れる |
| 5章 | 真実を知り、彼の七周期を撫でる | すべてを明かす。「呪いなら、あります。わたしに、です」 |
| 終章 | 眠りぎわ、初めての恋の自覚 | 一度だけ、名前を呼ぶ |
| 2周目 | —— | 絵本の続きを書き始める |
