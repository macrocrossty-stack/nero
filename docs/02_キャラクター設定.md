# キャラクター設定

## 少女（主人公）／仮名: ステラ

| 項目 | 内容 |
|---|---|
| 外見 | 10代半ば。白いワンピース。長い睫毛、淡い色の髪（スチルで確定）。裸足で目覚める |
| 性格 | 等身大で天然。『こばと。』『ARIA』の灯里系。一般常識はあるが、世界への驚きが新鮮 |
| 一人称 | わたし |
| 口調 | やわらかい。「〜ですね」「〜かなあ」。ネロには最初から敬語を使わない（距離の近さの表現） |
| 正体 | 古代人が作った対災厄用の調律装置。感情を持つことで暴走のリスクがあるが、黒豹との絆で「心」を得た |
| 立ち絵 | **基本なし**（没入感優先）。暴走時・重要シーンのみ表示 |

### 書き方のガイド
- 「知識はあるのに記憶（経験）がない」ちぐはぐさを台詞に出す。
  - 例: パンの焼き方は知っているのに、食べた記憶がない。
- 恐怖よりも先に好奇心が動く。怖がるのは「ネロが傷つくこと」に対してだけ。
- 地の文は少女の一人称。感覚描写（光・匂い・温度）を多めに。

## 黒豹（従者）／ネロ

| 項目 | 内容 |
|---|---|
| 外見（人間時） | 20代半ば。執事服。黒髪、金の瞳（黒豹と同じ瞳であることが伏線） |
| 外見（黒豹時） | 大型の黒豹。左肩に古傷（＝少女を庇った時の傷。人間時も同じ位置に傷がある） |
| 正体 | 元は古代の研究員/警備兵。少女を庇って瀕死になり、彼女の力で「黒豹」として固定された |
| 性格 | 寡黙、忠誠、自己犠牲。感情は瞳と行動にだけ出る |
| 呼称 | 少女を「お嬢様」と呼ぶ。**終章まで絶対に名前を呼ばない** |
| 口調 | 敬語を崩さない。「〜でございます」「なりません」。短文。 |
| 戦闘形態 | ①通常（人間・剣）→ ②本気（黒豹）→ ③禁忌（半獣半人。少女を怖がらせないため隠している） |

### 書き方のガイド
- 嘘はつかないが、真実を言わない。「呪い」という言葉は少女が言い出し、ネロは**否定しない**だけ（ミスリードの作法）。
- 献身は台詞でなく所作で描く（食事は先に毒見をしている、寝ずに扉の前にいる、等）。
- ユーモアは無自覚に。真顔でずれたことを言う。

## 敵：空の災厄（影の怪物）

| 項目 | 内容 |
|---|---|
| 正体 | 宇宙的侵食。意思はあるようで無い。「空の色が濁る」ことで接近がわかる |
| 現れ方 | 見る者の恐怖を反映した「影の怪物」。少女には獣に、ネロには人型に見える（＝ネロが最も恐れているのは「人間だった自分」という示唆） |
| 演出 | 出現時は画面の彩度を落とし、BGMを止めて環境音のみにする |

## 関係性の推移（章ごと）

| 章 | 少女→ネロ | ネロ→少女 |
|---|---|---|
| 1章 | 「かわいそうな呪われた人」保護欲＋好奇心 | 任務としての庇護（に見せた贖罪） |
| 2章 | 「わたしを守ってくれる人」信頼の芽生え | 力の兆しへの怯えと決意 |
| 3章〜 | 恋と呼ぶ前の独占欲。名前を呼んでほしい | 想いの自覚と封印 |
| 終章 | すべてを察して、それでも笑って眠る | 一度だけ名前を呼ぶ |
