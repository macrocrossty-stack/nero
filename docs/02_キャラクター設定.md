# キャラクター設定（改稿版）

## 少女（主人公）／仮名: ステラ（終章まで名前は非表示）

| 項目 | 内容 |
|---|---|
| 外見 | 10代半ば。白いワンピース。淡い色の髪（スチルで確定） |
| 暮らし | 森の屋敷で**ひとり暮らし**。料理も掃除も自分でこなす。「いつからここにいるか」を疑問に思ったことがない |
| 性格 | 等身大で天然。『こばと。』『ARIA』の灯里系。不安を強がりと笑いに変える癖がある |
| 一人称 | わたし |
| 口調 | やわらかい。「〜ですね」「〜かなあ」。ネロには最初から敬語を使わない |
| 記憶 | **喪失の自覚がない**。第2章「名前の穴」で初めて気づく（名前→両親→来歴の順に、無いことが判明していく） |
| 正体 | 古代人が世界の記憶と歌を注ぎこんだ器（調律装置）。調律のたび自分の記憶を燃やす。感情は共鳴を加速させる |
| 立ち絵 | 基本なし（没入感優先）。"豹変"時・重要シーンのみ表示 |

### 書き方のガイド
- 第1章の彼女は**何も欠けていない普通の女の子**として書く。孤独を寂しがらないことの異常さは、地の文の明るさで隠す。
- 「知識はあるのに経験の記憶がない」ちぐはぐさ（パンの焼き方は手が覚えているのに、はじめて焼く）。
- 記憶喪失の発覚後は、明るさが**演技を含む**ようになる。ベッドの下の「おぼえていることリスト」のような、こっそりした行動で不安を描く。
- 恐怖の対象は自分の正体よりも「ネロが傷つくこと」「この日常が終わること」。
- "豹変"（調律状態）では、一人称の地の文に無機質な文体が混入する（エンジン演出と連動）。

## 黒豹（従者）／ネロ

| 項目 | 内容 |
|---|---|
| 外見（人間時） | 20代半ば。執事服。黒髪、金の瞳（黒豹と同じ瞳＝伏線） |
| 外見（黒豹時) | 大型の黒豹。左肩に深い傷（第1章で少女が手当てする傷。人間時も同じ位置） |
| 登場 | 第1章で**傷ついた黒豹として拾われる**（絵本前編のなぞり）。夜のわずかな時間だけ人の姿に戻る、と少女には説明 |
| 傷の理由 | 毎夜、森の外周で影を狩って侵食を遅らせている。毎朝新しい傷が増えるのはそのため（中盤まで秘匿） |
| 正体 | 元は古代の研究員/警備兵。少女を庇って瀕死になり、彼女の力で「黒豹」として固定された。七度の周期をすべて憶えている |
| 性格 | 寡黙、忠誠、自己犠牲。感情は瞳と行動にだけ出る |
| 呼称 | 「お嬢様」。第2章で、名乗れない彼女への配慮として生まれる呼び方。**終章まで絶対に名前を呼ばない**（名前＝解放の鍵だから） |
| 口調 | 敬語。「〜でございます」「なりません」。短文。戦闘時のみ一人称が「俺」に揺れる |
| 戦闘形態 | ①人間・剣 → ②黒豹 → ③禁忌（半獣半人。少女を怖がらせないため隠している） |
| 絵本 | **『姫と黒猫の物語』の作者**。毎周期、目覚めた彼女のために書庫に置く。後編のページを破り取ったのも彼（クリア後判明） |

### 書き方のガイド
- 嘘はつかないが、真実を言わない。「呪い」は少女の言葉で、ネロは**否定しない**だけ。
- 献身は台詞でなく所作で（毒見、扉の前の気配、竈の火加減、絵本の綴じ目を直す手つき）。
- ユーモアは無自覚に。真顔でずれたことを言う（帽子パン等）。
- 彼の「お嬢様」は距離ではなく祈りである、と書き手は常に意識する。

## 敵：空の災厄（影の怪物）

| 項目 | 内容 |
|---|---|
| 正体 | 世界に「自分の形を忘れさせる」宇宙的侵食。太陽が消える・色が濁る・名前が失われる |
| 影の怪物 | 忘れられたものたちの成れの果て。「思い出されよう」ともがくため、見る者の恐怖を映す。恐れるほど形を得て育つ |
| 音 | 災厄側＝ガラスの**軋む**音 ／ 少女の力＝ガラスの**共鳴**音（清濁の対比、毎回同一SE） |
| 演出 | 出現時はBGM停止→環境音のみ→彩度ダウン |

## 関係性の推移（章ごと・改稿版）

| 章 | 少女→ネロ | ネロ→少女 |
|---|---|---|
| 1章 | 拾った「絵本みたいな」黒豹。保護欲＋好奇心 | （豹として）沈黙の庇護 |
| 2章 | 「よるのひと」への信頼。名乗れない自分を受け止めてくれた人 | 名前の穴に触れさせない防衛。力の兆しへの怯え |
| 3章 | 恋と呼ぶ前の独占欲。彼の夜の傷の秘密へ | 想いの自覚と封印 |
| 4章 | 真実への接近と"豹変"の恐怖 | 禁忌の解放。隠し事の限界 |
| 終章 | すべてを察して、それでも笑って眠る | 一度だけ、名前を呼ぶ |
