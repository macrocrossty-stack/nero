# UI・画面構成案

方針: **没入感優先**。UIは「絵本と硝子」をモチーフに、常時表示物を最小にする。

## 実装状況（エンジンv3）

| 機能 | 状態 |
|---|---|
| タイトル画面（はじめから／つづきから） | ✅ 実装済み |
| オートセーブ（クリック待ちごとにlocalStorage保存） | ✅ 実装済み |
| バックログ（LOGボタン／Lキー、直近300件） | ✅ 実装済み |
| 立ち絵レイヤー（[chara]/[chara_hide]、表情差分対応） | ✅ 実装済み（仮素材: 影絵＋金の瞳） |
| 文字あふれ対策 | ✅ フォント縮小＋ウィンドウ拡大＋縦スクロール保険 |
| ヒロイン立ち絵 | 1周目の覚醒（第5章）から表示する方針。素材待ち |
| AUTO / SKIP / コンフィグ | 未実装（次フェーズ） |

## 立ち絵の設計ルール（決定）

**立ち絵を持つ＝彼女の世界で「かたち」を持つ存在。**
- 全編を通して立ち絵を持つのは**ヨルだけ**（人間形態・黒豹）。
- **村人などのモブは立ち絵なし**（声だけの存在）。没入感の維持とコスト削減を兼ねる。
- 彼女自身は第5章の"豹変"まで立ち絵なし。初めて表示されるのが**「調律機関」として形を与えられた瞬間**であること自体が演出（girl_tuning.png）。
- girl_normal.png は終章専用に温存。

## 立ち絵の仕様

- 配置: 画面右寄せ固定（右5%、高さ86%）。ドロップシャドウ付き
- ファイル: `game/data/fgimage/`（yoru_normal / yoru_smile / yoru_battle / yoru_panther）
- 本番素材も同ファイル名で差し替えれば動く。推奨サイズ: 高さ1000px以上、透過PNG
- ヒロインは第5章の"豹変"シーンが初出し（それまで姿を見せないことが没入感と正体隠しに効く）

## 基準

- 解像度: 1280×720（16:9）
- フォント: 明朝系（しっぽり明朝 / 源暎こぶり明朝 など商用可のもの）
- 基調色: 深い青 `#0a0e14` / 文字 `#e8e4da` / アクセント（金の瞳の色）`#c9a86a`

## メッセージウィンドウ

- 画面下部 約28%、左右に余白。背景は黒 65% + 上端に1pxの金ライン
- 名前欄はウィンドウ左上に小さく。**少女の話者名は表示しない**（一人称地の文と統一し、「名前」への意識をラストまで温存する）
- 通常パートは下部ウィンドウ、**寓話パートのみ全画面テキスト**（縦中央、行間広め）に切り替え、絵本らしさを出す

## 常時表示UI

- 右下に小さく「AUTO / SKIP / LOG / MENU」の4つのみ。ホバーで点灯
- クリック待ちカーソル: 金色の小さな星（テーマ対応）。寓話パートでは頁角の折れ

## メニュー（MENU展開時）

セーブ / ロード / コンフィグ / タイトルへ。半透明黒のオーバーレイに縦並び。
コンフィグ項目: 文字速度 / オート速度 / BGM・SE音量 / 既読スキップ

## タイトル画面

- 一枚絵: 眠る少女と寄り添う黒豹のシルエット（仮: bg_forest_night に月）
- メニュー: はじめから / つづきから / コンフィグ。画面下部に横並び、小さく
- **周回要素**: クリア後は「はじめから」の文言が「もういちど」に変わる（ループエンドの余韻。実装は終章と同時で良い）

## 演出ルール（スクリプト共通）

| 事象 | 演出 |
|---|---|
| 場面転換 | 黒フェード 800ms（`[bg time=800]`） |
| 時間経過 | 白フェード 1200ms |
| 侵食の接近 | BGM停止 → 環境音のみ → 画面彩度ダウン（仮: bg_sky_corrupt へ差し替え） |
| 少女の力 | ガラスの共鳴SE + 画面全体を白 20% フラッシュ（毎回同じ組み合わせで統一） |
| 寓話パート | 全画面テキスト + オルゴールBGM + 頁めくりSE |
