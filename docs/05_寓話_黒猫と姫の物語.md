# 劇中絵本『姫と黒猫の物語』（推敲版・前後編分割）

原文: Notion「📖冒頭絵本」。作者様の文章を土台に、リズムと表記を整えた推敲版。
**前編**＝ゲーム内で普通に読める部分（ハッピーエンドに見える）。
**後編**＝破りとられた隠しページ。中盤以降に発見される。末尾にループ暗示の加筆あり。

設定: この絵本の作者は実はヨル（クリア後に判明）。目覚めるたび記憶を失う彼女に、
「黒い獣を怖がらず、愛し方を思い出してもらう」ために毎周期、書庫に置いている。
後編を破り取ったのもヨル（結末を知られたくないため）。
第5章で祭壇に原稿が見つかる（筆跡＝旅のあいだ見慣れた彼の字）。
2周目ラストで、彼は七周期で初めて最終ページの続きを書き始める。

---

## 前編（ゲーム内絵本・プロローグで朗読）

むかしむかし、ある遠い国に、ひとりぼっちのお姫さまがいました。

お姫さまは高い塔に暮らして、まいにち、窓から見える深い森をながめて過ごします。
お友だちも、お話を聞いてくれるひとも、いません。
小鳥のさえずりと、やさしい風のにおいだけが、お姫さまの心を、そっとなぐさめてくれるのでした。

ある静かな日のこと。塔の下から、か細い声が聞こえました。

「にゃあ……にゃあ……」

草むらのあいだで、傷ついた黒猫が、夜のかけらのように小さくふるえていたのです。

お姫さまは急いで塔を駆けおりて、黒猫をそうっと抱きあげました。
毛なみは宵闇のように黒く、瞳はお星さまのようにきらきらとかがやいて、
からだは、濡れた石のように冷たくなっていました。

お姫さまは黒猫を塔へつれて帰り、やさしく傷の手当てをしてあげました。

それから黒猫はすっかり元気になって、ふたりは、いつもいっしょでした。
ひとつのお皿をなかよく分けあい、お庭の小道をならんで歩き、
夜になるとお姫さまの胸の上で、黒猫は小さな小さな寝息をたてるのです。

黒猫は、お姫さまの、生まれてはじめてのお友だちでした。

けれども——猫のいのちは、ひとのいのちより、短いもの。

季節がみっつめぐったころ、黒猫はよわって、お姫さまの腕のなかで、静かに目を閉じました。
お姫さまは深い深いかなしみに沈みました。
涙はことばよりも先にあふれて、黒猫のひたいへ、そっと落ちていきました。

そしてお姫さまは、黒猫に、お別れのくちづけをしたのです。

——そのときでした。

黒猫のからだがまばゆい光につつまれて、気がつくと、そこには、
ひざまずく、ひとりの美しい騎士のすがたがありました。

「姫君。わたしは呪いによって、猫のすがたに変えられておりました。
　あなたのやさしい心と、深い愛が、その呪いを解いてくださったのです」

お姫さまは涙をぬぐって、花のようにほほえみました。

騎士は、黒猫のときと同じようにお姫さまに寄りそい、
お姫さまは、黒猫のときと同じように、騎士に美しい歌をおくりました。
ふたりのお部屋には、朝のお日さまのような、あたたかな光が、まいにちさしこみました。

こうしてふたりは、いつまでも、いつまでも、しあわせにくらしました。

——おしまい。

※演出: 「おしまい」の次を開こうとすると、綴じ目に破りとられたページの根もとが残っていることがわかる。

## 後編（破りとられたページ・中盤以降に発見）

ほんとうのお話には、つづきがあります。

ある日、空から、お日さまが消えてしまいました。

太陽はのぼらず、森はざわめき、大地は日ごとに凍りついていきます。
お花は枯れ、小鳥たちは飛び去り、世界はまるで夜のまま眠ってしまったかのように、
しいんと、静まりかえりました。

騎士は剣を手にとり、お姫さまを守るため、暗い森の奥へと歩きだしました。

寒さは骨を刺し、道は雪にとざされました。
騎士はなんども立ちどまり、息をととのえ、また歩きました。
お姫さまの笑顔を胸に。黒猫だったころの、あたたかなぬくもりを心に抱いて。

やがて——騎士の足は、とまりました。
雪の上にしずかに身をよこたえ、凍てつく夜のなかで、そっと目を閉じてしまったのです。

そのころ、塔の窓辺で、お姫さまは、ひとり祈っていました。

「どうか、どうか、あの方に光を……」

けれど祈りのことばはひそやかで、遠い森の騎士のもとまでは、とてもとどきません。

ひとしずく、またひとしずく。
お姫さまの頬から、真珠のような涙がこぼれおちました。

やがて涙は、頬からこぼれるだけでは足りなくなって——
指さきがそっとほどけ、絹のような髪がやわらかくほどけ、
心のいちばん奥のまんなかだけをのこして、
お姫さまは、透きとおるお水のかけらになっていきました。
かけらは天の光にとけて、きらめく無数の雫となったのです。

雫は音もなく空へのぼり、冷たい雲のなかへしずかにまじって、
やがて雨となって、大地へ帰っていきました。

雨は森をやさしくうるおし、凍てついた氷をとかし、
長いあいだ眠っていた小川を、もういちど目ざめさせました。
長い長い夜のはてに、重い雲がうすらいで、暗い空のはしっこに、希望の金いろがほのかにうまれ——
とうとう、待ちわびたお日さまが、帰ってきたのです。

騎士の眠っていた雪のあとには、まっしろなお花がひとつ、ひっそりと咲いていました。
そよ風が吹くたびに、お花は祈るように小さくうなずいて、ゆらゆらとゆれました。
その白さは、はるかな空の雲のいろのようでもあり、
天から降った、雨のひとしずくのようでもありました。

ふたりのすがたは、もう、どこにもありません。

——けれど。

白いお花のそばには、いつからか、一匹の黒い猫が、すわっています。

お花が咲くたびに、黒猫は「はじめまして」と鳴くのです。

なんども。

なんども。

## 原文からの主な変更点

1. **前後編に分割**し、前編の末尾に「いつまでも、いつまでも、しあわせにくらしました。——おしまい。」を追加（ハッピーエンド偽装の綴じ目）。
2. **後編冒頭に「ほんとうのお話には、つづきがあります。」を追加**（発見シーンの一行目として機能）。
3. **末尾にループ暗示を加筆**（白い花のそばの黒猫、「はじめまして」と鳴く）。騎士＝黒猫の再生とループ構造を、本編ラストで回収。
4. 表記ゆれ・誤字の修正（「なっていきした」→「なっていきました」等）、ひらがな率を絵本らしく統一、長文のリズム分割。
5. 象徴表現（夜のかけら・真珠の涙・心のいちばん奥のまんなか・白い花）は**すべて原文のまま保持**。

## 本編との対応表

| 絵本 | 本編 |
|---|---|
| 塔でひとりぼっちの姫 | 屋敷でひとり暮らしの少女（孤独の自覚がない分、より静かに怖い） |
| 傷ついた黒猫を拾う | 傷ついた黒豹を拾う（第1章。プレイヤーに「王子に戻す話」と思わせる） |
| くちづけで呪いが解ける | ミスリードの核。「愛が呪いを解く物語」だと信じさせる |
| お日さまが消える | 空の災厄（世界が自分の形を忘れていく侵食） |
| 姫が雨になり世界を救う | 少女の調律＝記憶を燃やして世界を歌い直す |
| 白い花と黒猫の「はじめまして」 | ループエンド。目覚めるたびの再会 |
| （枠外）この絵本の作者 | ヨル。2周目ラスト、破れた最終ページの続きが書き始められる——「つづく」 |
