# 劇中寓話「黒猫と姫の物語」（ドラフト）

> ⚠️ **注意**: Notionの企画書に「フルバージョン作成済み」とあった完成版テキストは、過去チャット内にあり本リポジトリからは参照できませんでした。
> 以下は脚本を進めるための**ドラフト版**です。お手元の完成版があれば、このファイルと `game/data/scenario/prologue.ks` の該当部分を差し替えてください。

むかしむかし、遠い国のそのまた向こうに、
ひとりの姫がおりました。

姫はふしぎな子でした。
姫が歌えば嵐はしずまり、
姫が眠れば、夜空の星はみな、安心してまたたくのです。

けれど姫は知りませんでした。
じぶんの歌が、空を支えていることを。

姫のそばには、一匹の黒猫がおりました。
黒猫はいつも、ひとあし後ろを歩きます。
姫が笑えば尻尾をゆらし、
姫が泣けば、だまってひざにのりました。

「ねえ、おまえの名前を教えて」

姫がたずねても、黒猫はこたえません。
猫はしゃべれないのだと、姫は思っておりました。

——ある夜のこと。
空のはしっこが、墨をこぼしたように濁りました。
星がひとつ、またひとつ、こわい夢を見て消えていきます。

姫は歌いました。ひと晩じゅう、歌いました。
歌いおわると、姫はとてもねむくなって、
ながい、ながい眠りにつくのです。

眠った姫は、ぜんぶ忘れてしまいます。
嵐のことも、星のことも。
黒猫のことも。

それでも黒猫は、寝台のそばを離れません。
百年でも、二百年でも。
姫がつぎに目を覚まして、
「はじめまして、きれいな猫ね」と笑うその朝を、
なんどでも、いちばん幸せな朝として迎えるために。

さいごの夜、眠りにつく姫の耳もとで、
黒猫はたったいちど、ひとのことばを話しました。

なんと言ったのかは、だれも知りません。

そのさきのページは、
やぶれてしまって、読めないのです。

——おしまい。

## 使用箇所と演出メモ

| 使用箇所 | 使い方 |
|---|---|
| プロローグ | 全文を全画面テキスト+オルゴールで朗読。「やぶれてしまって〜」まで見せてタイトルへ |
| 第1章・書庫 | 実物の絵本として登場。最後のページが破れていることを少女が発見する |
| 終章 | ネロが名前を呼ぶ場面に「黒猫はたったいちど、ひとのことばを話しました」を重ねる。破れたページの内容＝いま起きていること、として回収 |

## 伏線対応表

| 寓話 | 本編の真実 |
|---|---|
| 姫の歌が空を支える | 少女＝対災厄用調律装置 |
| 黒猫は名前をこたえない | ネロが「お嬢様」としか呼ばない |
| 墨をこぼしたような空 | 空の災厄（宇宙的侵食） |
| 眠るとぜんぶ忘れる | 記憶リセットの不可避性 |
| ひとのことばを話す黒猫 | 黒豹の正体は人間 |
| 破れた最後のページ | ループの「その先」＝ゲームの結末そのもの |
