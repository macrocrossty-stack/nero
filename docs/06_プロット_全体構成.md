# 全体プロット（決定版構成）

2026-07-09〜10 打ち合わせの決定を反映した、プロローグ〜クリア後〜2周目までの全体構成。
第1〜2章の本文執筆はこのプロットの承認後に開始する。

凡例: ★＝伏線 ／ ⚙＝エンジン演出 ／ 🌱＝プレイヤー選択の収集ポイント（終章で燃やす）

## 決定事項の要約

- ヒロインの記憶喪失は**最後の最後まで完全に隠す**。「名前が思い出せない」等の明示ギミックは使わない。彼女の日常は欠けて見えず、完結して見える。
- ミスリードは三重: ①彼女は屋敷の家のお嬢様（偽の来歴をプレイヤーに自作させる）②これは呪いを解く物語（解呪クエスト）③これは静かな童話（ジャンル偽装）。
- 黒豹の名は**ヨル**（今周期に彼女が命名。「夜のあいだだけ人に戻れるから」）。過去周期の名前（クロ、ノワール、ヨイ、ネロ…）は旅先の言い伝えに埋める。
- 文明レベルは「**材質は童話・意味はSF**」。古代技術は聖堂・楽器・天文台の姿でしか描かない。異質さは最深部「星の部屋」一箇所のみ。
- 終盤は反転の連鎖: ①呪いはなかった → ②呪いはあった（ヨル＝種火の役目）→ ③プレイヤーが装填していた（選択の焼却・セーブ消去）。
- 種はひとつ残る（完全リセットにしない）。クリア後「もういちど」。
- 2周目はヨル視点レイヤー＋意味反転エンド（絵本の続きを書き始める・「つづく」）。

## 第1章「ひとりのお屋敷」

**狙い**: 静かな童話として開幕。彼女＝等身大で礼儀と常識のある「童話の中の子」。謎はすべて黒豹側に積む。

1. **朝のひとり暮らし**。パンを焼き、掃除し、庭に水をやる。明るく、満ち足りて、何の穴もない日常として書く。🌱（焼くパンの種類）
   - ★違和感の種（彼女は気に留めず、地の文も立ち止まらない）: カップが2組／止まった大時計（「動いてるところ、見たことない気がする」）／埃のない一室／廊下の肖像画の顔だけ褪せている／彼女は思い出話を一度もしない（気づけない設計でよい）
   - 日課の締めは書庫。お気に入りの絵本。最後のページが破れているのは「昔からそう」★
2. **夜の音**。森から獣の悲鳴と、ガラスの軋む音★（災厄側の音）。布団にもぐる、普通の女の子の怖がり方。
3. **黒豹を拾う**。翌朝、庭の茂みに傷ついた黒豹。左肩の裂傷。恐怖より先に絵本が浮かぶ——「絵本と同じだ」。手当て。豹は牙を向けず、包帯を巻きやすいよう体をずらす★。
4. **看病の日々**。朝ごとに結び目の替わる包帯★／夜中の人の気配★／言葉を解する金の目／絵本の読み聞かせに目をそらす豹★／「この子が来てから朝ごはんがおいしい」（理由は考えない）★。🌱（豹に読んでやる本の選択）
5. **章末**。夜半、豹の姿がない。月光の広間に執事服の男の後ろ姿、左肩に彼女の結び方の包帯★。雲が月を隠し、闇が戻ると豹が伏せているだけ。「……夢？」で幕。

## 第2章「よるのひと」

**狙い**: 解呪クエストの成立。呼称と命名をここで発生させる（記憶喪失の匂いは一切出さない）。

1. **対面**。寝たふりで待ち伏せ→現行犯（コミカルに）。執事服の青年、金の瞳。
   - 彼の名乗り:「昔、このお屋敷にお仕えしていた者でございます」→ **「お嬢様」呼びが自然に発生**。プレイヤーは「彼女＝この家の娘」と自作補完する★。
   - 「呪いで、夜のわずかな間しか人に戻れません。……名前も、呪いに喰われました」→ **黒豹側の記憶喪失（フリ）**。ただし名前の喪失だけは真実★。
2. **命名**。「じゃあ、わたしが付けてもいい？　——夜のあいだだけ人に戻れるんでしょ。だから、ヨル」。彼が一瞬だけ言葉を失う★（七周期ぶん、この瞬間を繰り返してきた男の顔）。
3. **ふたりの日常**。毒見★、扉の前の気配★、真顔のずれた発言（帽子パン）。毎朝、癒えたはずの体に新しい傷★（夜の狩り。はぐらかされる）。
4. **空の澱みと初襲撃**。鳥が消える→油膜の空→庭に影の怪物。昼なのに人化して剣で応戦（「夜だけ」も嘘？★）→劣勢→**黒豹への変身**を目撃→影が彼女に迫る→**力の芽生え**。ガラスの共鳴音、白い光、影がほどけて消える。
5. **誤読の固定**。彼女自身が解釈する——「これ、呪いを解く力なんじゃない？」。ヨルは長い沈黙ののち:「……呪いを解く鍵は、あなたなのかも、しれません」（彼の創作クエストの開始宣言。そして全層で真実になる台詞★）。
6. **章末・研究日誌**（プレイヤーのみ）。書庫の隠し棚、黒革の手帳、古代文字⚙（彼女には読めない）。「空の澱み、周期どおりに再来」「器の自律共鳴」「感情は共鳴を加速させる」「これは七度目の記録である」。手帳を戻した彼が、**慣れた手つきで絵本の綴じ目を直す**★。

## 第3章「かつてのひとをさがして」

**狙い**: 解呪クエスト本格化＝屋敷の探索。関係が深まるほど「解呪が進む」偽の証拠。絵本後編の発見。

1. **手がかり探し**。「呪われる前のヨル」の痕跡を屋敷で探す（彼の提案でもある＝彼の舞台装置）。
   - 埃のなかった一室＝「使用人の部屋」（第1章★の回収に見せた、新たなミスリード）
   - 顔の褪せた肖像画、書きかけの手紙、古い制服……ぜんぶ彼が毎周期置いてきた小道具
2. **偽の証拠**。絵本のとおりの「おまじない」を試す——ひとつの皿を分ける、歌をおくる🌱（おくる歌の選択）。試すたび、彼が人でいられる時間が**実際に延びる**（真実は彼女の力の共鳴）。「呪い、解けかけてるんだよ！」
   - くちづけの頁で悶絶して本を閉じる（恋の自覚はまだ無い。ただ「なんか、へん」とだけ）★
   - 躱した直後、豹の尻尾が一度だけ床を打つ★（彼の抑圧の所作）
3. **絵本後編の発見**。探索の副産物として、破りとられたページを発見（彼の想定外——七周期で初めての事故★）。「ほんとうのお話には、つづきがあります」——寂しい結末に彼女が沈む。ヨルの顔が一瞬だけ強張る。
4. **地図**。痕跡のなかから古い地図。森の外、「呪いの始まりの場所」を示す印。「行こう。ヨルの呪い、ぜんぶ解こう」——**旅の始まり**。
5. **章末**（プレイヤーのみ）。夜、荷造りをする彼の独白気配。日誌に一行:「今回も、旅になった」★。

## 第4章「ちいさな旅」

**狙い**: 世界の質感と災厄の実在感。ループの物証を風景に隠す。情緒の成長＝装填の進行。

1. **森を出る**。彼女、生まれて初めて（だと思っている）屋敷の外へ。世界は美しく、空の澱みは森の外からも見える——広がっている。
2. **ひとつめの村**。「白い娘と黒い獣」の言い伝え——数百年前の話として。獣の名は「**クロ**」★。彼女は「わたしたちみたい」と笑う。プレイヤーは神話モチーフだと流す。
   - 忘却の被害の点描: 名前を失くした橋、顔を思い出してもらえない肖像画の家★（屋敷の褪せた肖像画と同じ現象）
3. **ふたつめの町**。同じ言い伝え、別の時代、獣の名は「**ノワール**」★。細部が違うのに、白い娘はいつも「よく眠る子」★。🌱（市で買うもの・彼に贈る小物）
4. **夜の宿**。情緒の成長を刻む章。名前のつかない、あたたかくて少し苦しいもの（恋の未満）。ヨルは一定の距離を崩さない——プレイヤーには「呪いを解きたいはずなのに、なぜ一番それらしい距離を避ける？」という違和感が蓄積★。
5. **街道の襲撃**。過去最大の影。黒豹でも押される→一瞬、**半獣の禁忌形態のシルエット**（彼女は気絶していて見ない。プレイヤーだけが見る）★。
6. **章末**。地図の示す終点が、大きな弧を描いて**自分たちの森の裏側**を指していることに気づく。「呪いの始まりの場所って……うちの、下？」

## 第5章「ほしのへや」

**狙い**: 反転の連鎖。材質は童話のまま、意味だけがSFに反転する章。

1. **遺跡へ**。森の裏、崖の下の扉。中は——大聖堂だった。パイプオルガンに似た塔の心臓、真鍮の天球儀、星図盤。冷たい機械はひとつもない。ただ、どれも「知っている気がする」★。
2. **反転①「呪いは、なかった」**。祭壇に、あの絵本の**原稿**。作者の筆跡は、旅のあいだ見慣れた彼の字。解呪クエストの手がかり、屋敷の痕跡、地図——ぜんぶ、彼が置いたもの。
3. **彼女の"豹変"**⚙。遺跡が彼女に応えて起動する。選択肢が表示され、**選ぶ前に勝手に選ばれる**。独白に等幅の無機質な文が混ざる。ずっと空欄だった名前欄に「**調律機関**」と表示される。プレイヤーの「操作している」という前提が崩れる。
4. **反転②「呪いは、あった」**。真実の開示（彼の口から、静かに）: 彼女は世界に自分を思い出させる歌い手。記憶は代金。彼は最初に認識された人間＝**種火**。システムは彼を手放さない。「呪いなら、あります。……わたしに、です」。
5. **災厄の本体**。空が割れるのではなく、**空が「空」という名前を忘れていく**。星の部屋（プラネタリウムのような最深部、彼女が生まれた場所）へ。禁忌形態の解放、しんがりの戦い。
6. **選択**。彼女は全部を理解した上で——怒りも取り乱しもせず、「ずっとひとりで持ってたんだね、これ」と彼の七周期を撫でる。そして歌うことを選ぶ。

## 終章「ねむりひめ」

1. **調律**⚙。燃える記憶の数え上げ——汎用テキストではなく、**プレイヤーが選んだもの**（🌱の収集結果）を一つずつ。「はじめて焼いた、帽子みたいなパン」「あなたに読んだ、よだかの星」……
2. ⚙バックログの過去テキストが実際に消えていく。画面の既読履歴が白紙になっていく。
3. **名前呼び**。すべてが燃え尽きる寸前、彼が初めて、彼女の名を呼ぶ（真の名＝解放の鍵。全編でこの一度だけ、名前欄に彼女の名前が表示される）。
4. **恋の自覚**。「そっか。わたし、あなたのこと——」眠りが連れていく。言葉は完成しない。
   最後の吐息で:「つぎに会ったら、こんどは、あなたにほんとうの名前をあげるね」
5. **絵本後編の黒猫の頁**がオーバーレイ。「なんども。なんども。」
6. ⚙**セーブデータ消去**。ゲームは「何も残っていません」という顔をする。——ただし種がひとつ残る（不可視フラグ）。
7. エンドロール。明けて、**8回目の目覚め**: 第1章と同じ書き出しが始まり、庭の黒豹を見て——「……おかえりなさい？」で暗転。
8. タイトル画面。「はじめから」が「**もういちど**」に変わっている。

## 2周目（ヨル視点レイヤー）

「もういちど」で開始。本編と同じシナリオに、以下を重ねる:

- **彼の独白レイヤー**⚙: 同じ場面に、彼の内心が差し込まれる（命名の瞬間、毒見、尻尾が床を打った夜）。
- **挿入夜景**: 彼女が眠っている間の十数シーン（外周の狩り、竈の火加減、絵本の綴じ目直し、日誌、荷造り）。
- **1周目で隠したもの**の開示: 禁忌形態の全景、日誌の全文、過去周期の彼女たちの断片（旅の言い伝えの真実）。
- **エンドの延長**: 暗転の先が続く。眠った彼女。ひとりの広間。彼は書庫で絵本を開き、七周期で初めて、**最終ページの続きを書き始める**。内容は見せない。「つづく」とだけ表示して、幕。
- 8回目の「……おかえりなさい？」が、今度は**彼の七百年に初めて起きた異変**として刺さる。ループは終わらない。だが「終わらないこと」が牢獄から約束に反転する。
