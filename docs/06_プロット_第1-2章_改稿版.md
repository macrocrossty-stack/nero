# 改稿プロット — 第1章・第2章

方針（2026-07-09 打ち合わせで決定）:
- ヒロインは**屋敷でひとり暮らし**。記憶喪失の**自覚がない**（「いつからここにいるか」を疑問に思ったことすらない）。
- 序盤、秘密を握る「不思議な存在」は**黒豹側**に見えるようにする。
- 記憶の欠落への気づきは、力の芽生えとセットで**恐怖として**訪れる。
- 絵本前編（王子に戻る話）を先に読ませ、「これは猫を王子様に戻す物語だ」とプレイヤーに思い込ませる。

凡例: ★＝伏線を仕込む箇所 ／ ⚙＝エンジン演出（自作エンジンならでは）

## プロローグ「えほん」

絵本『姫と黒猫の物語』**前編のみ**を全画面テキストで朗読。
「こうしてふたりは、いつまでも、いつまでも、しあわせにくらしました。——おしまい。」で本を閉じ、タイトルロゴ → 第1章へ。

## 第1章「ひとりのお屋敷」（旧・邂逅）

### 1. 朝のひとり暮らし
- 少女の一人称で、森の屋敷での朝。パンを焼き、掃除をし、庭に水をやる。楽しく、等身大に、**何の不安もなく**書く。
- 「ひとりであること」を彼女は寂しがらない。そういうものだと思っている。★（この「疑問のなさ」自体が異常だが、地の文は明るいので読者は流す）
- ★違和感の種を無言で置く（彼女は気に留めない）:
  - 食器棚に、カップが**2組**ある（彼女は「予備」と思っている）
  - 広間の大時計が止まっている
  - 使っていないのに埃のない部屋がひとつある
- 日課の締めは書庫。お気に入りの絵本『姫と黒猫の物語』。「もう何度読んだかわからない」★（実は毎周期これで"教育"されている）。最後のページが破れているのも「昔からそう」。

### 2. 夜の音
- 夜、森のほうから聞いたことのない音（獣の悲鳴と、ガラスが軋むような音★＝災厄側の音）。
- 怖くて布団にもぐる。ここは普通の女の子の怖がり方で。

### 3. 黒豹を拾う
- 翌朝、庭の茂みで傷ついた黒豹を発見。左肩に深い裂傷。
- 恐怖より先に絵本が浮かぶ——「にゃあ、って鳴いた……わけないか」。**絵本と同じだ**、と胸が高鳴る。
- 手当て。豹は牙も爪も向けない。それどころか、彼女が包帯を巻きやすいように体をずらす★。

### 4. 看病の日々と「不思議」の集積（すべて黒豹側に謎を寄せる）
- 朝ごとに、包帯の結び目が**自分の結び方と違う**ものに替わっている★
- 夜中、階下にひとの気配。降りてみると誰もいない。豹は定位置にいる——が、床に**人の足あと**らしき濡れ跡★
- 豹の金色の目は、彼女の言葉を明らかに理解している
- 彼女は豹に絵本を読み聞かせる。「あなたも王子様だったりして」と冗談を言うと、豹が**目をそらす**★
- このあたりで彼女の独白として「そういえば、この子が来てから、朝ごはんがおいしい」→ 理由は考えない★（ネロが夜中に竈の火加減を直している）

### 5. 章末「よるのひと」
- 夜半、目が覚める。豹がいない。
- 探して降りた広間、月光の中に**執事服の男の後ろ姿**。左肩に、彼女の結び方のままの包帯★。
- 声を出せずにいるうち、雲が月を隠し、闇が戻ると——広間には豹が伏せているだけ。
- 「……夢?」で幕。第1章・了。

**章の狙い**: プレイヤーの疑問を100%黒豹に向ける。少女は「観測者＝プレイヤーの分身」として完全に信頼させる。

## 第2章「よるのひと」（旧・力の芽生え）

### 1. 対面
- 翌晩、彼女は寝たふりで待ち伏せ（等身大の行動力。ちょっとコミカルに）。
- 深夜、階下に降りた「ひと」を現行犯で捕まえる。執事服の青年。金色の目。
- 「絵本と同じ！　呪いが解けたの!?」→ 青年は長い沈黙ののち「……解けては、おりません。人でいられるのは夜のわずかな間だけ」。**呪いという言葉を否定しない**（ミスリード成立の瞬間）。
- 名前を聞く。「——ネロ、と。昔、そう呼ばれておりました」

### 2. 名前の穴（記憶喪失の発覚）
- 「わたしの名前はね——」と名乗ろうとして、**出てこない**。
- 名前だけではない。両親の顔。ここへ来た日。屋敷の前の記憶。**ぜんぶ、ない**。
- ここで初めて、彼女の一人称の地の文が揺れる（改行が乱れる、同じ問いの反復）⚙。第1章の明るさとの落差で、怖さを最大化。
- パニックになりかけた彼女に、ネロは静かに——「では、わたしがお呼びする間だけ。『お嬢様』と。……お名前は、きっと、しかるべき時に思い出されます」★（お嬢様呼びの発生に理由がつく）
- 彼女は泣きそうになりながら、それを笑いに変える。「お嬢様って柄じゃないんだけどなあ」——この強がりが、彼女の魅力の核。

### 3. ふたりの日常
- 夜だけ人に戻る同居人ができる。食事の毒見★、扉の前の気配★、真顔のずれた発言（帽子パン等のユーモア）。
- ★毎朝、癒えたはずの豹の体に**新しい傷**が増えている。彼女は「夜のうちに森で何かしてるの?」と聞くが、はぐらかされる（＝彼が毎夜、外周で影を狩って侵食を遅らせている。判明は後章）。
- 彼女は自分の記憶の穴を、明るさでごまかしながら、こっそり数えはじめている★（ベッドの下にメモ。「おぼえていること」のリストが、悲しいほど短い）。

### 4. 空の澱みと襲撃
- 鳥の声が消える。空の端に油膜のような澱み。⚙BGM停止→環境音のみ。
- 庭に影の怪物。**第1章2節の「夜の音」と同じ音**★で襲来を予告。
- ネロ、昼なのに人化して剣で応戦（→「夜だけ」も嘘?★）。劣勢。彼女の目の前で**黒豹に変身**（正体の一部開示）。
- 影はちぎれても消えず、彼女に迫る。

### 5. 力の芽生え
- 世界が「鳴る」。ガラスの共鳴音、手のひらから白い光。影は墨が水に溶けるようにほどけて消える。
- ⚙ここで一瞬だけ、メタ演出の**種**を蒔く: 彼女の独白の文体が1行だけ変わる（「——共鳴。出力最小。」のような等幅・無機質な一行が挟まり、彼女は「いまの、なに?」と自分の思考に戸惑う）。本格的な"操作の乗っ取り"は後章までとっておく。
- ネロが光る手を両手で包んで隠す。「なんでも、ありません」。その手が震えている。

### 6. 章末・研究日誌（プレイヤーだけの場面）
- 夜更け、書庫の隠し棚。ネロが黒革の手帳を開く。古代文字——**少女には読めないが、プレイヤーには読める**⚙。
- 内容: 空の澱みの再来周期 / 「器」の自律共鳴 / 感情は共鳴を加速させる / 「これは七度目の記録である」。
- 手帳を戻したネロが、ついでのように絵本を手に取り、**慣れた手つきで綴じ目を直す**★（作者＝ネロの一粒目の示唆）。
- 「……申し訳、ございません。俺はまた、あなたの朝を、祈ってしまう」で幕。

**章の狙い**: ミスリード（呪い）を固定しつつ、「記憶がない」恐怖を彼女とプレイヤーが同時に体験する。謎の重心はまだ黒豹側にあるように見せる。
