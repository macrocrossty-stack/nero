# 音楽・効果音仕様書

## 方針（決定済み）

- **世界観**: 灰羽連盟系オールドファンタジー。アコーディオン、弦、木管、オルゴール、ピアノ、遠い鐘。**電子音・シンセ質感は禁止**（「材質は童話」に対応）
- **AI作曲: 問題ありません**。Suno / Udio 等の利用を想定する場合、**商用利用可のプラン・規約であることだけ**確認してください（無料プランは商用不可のサービスが多い）。クレジット表記義務の有無も規約でチェック
- **指定方法**: 下の曲別仕様表をそのままAI作曲のプロンプト素材にできるよう書いています。**雰囲気の最終判断はお任せします**——迷ったら候補を2〜3曲もらえれば、シーンに当てて選定・実装します

## ファイル形式

| 用途 | 納品形式 | 実装形式 |
|---|---|---|
| BGM | **WAV または 320kbps MP3**（ループさせたい曲はループ位置メモ付きだと最高） | こちらで **m4a(AAC)** に変換して実装（Safari含む全ブラウザ対応。oggはSafari非対応のため不採用）。ループのつなぎ目はエンジン側でWeb Audioを使いギャップレス化します |
| SE | WAV | 同上 |

- 尺の目安: BGM 1分〜1分30秒あれば十分（ループ前提）。フェードアウトは**入れない**でください（ループが切れるため）
- 音量はこちらでノーマライズするので気にしなくてOK

## 実装の流れ

1. 素材をリポジトリの `game/data/bgm/`・`game/data/se/` に置く（またはチャットで受け渡し）
2. こちらで形式変換（m4a）＋シナリオ内のコメントアウト済み `[playbgm]` を有効化＋音量調整
3. プレビューで一緒に確認 → 差し替え調整

## エンジン対応状況（実装済み）

再生タグはエンジン(`preview/player.js`)に実装済み。**音声ファイルが未配置でもエラーにならず無音で進行**するので、シナリオのコメントを外すだけで鳴るようになります。

| タグ | 動作 |
|---|---|
| `[playbgm storage="bgm_daily.m4a"]` | BGM再生（ループ・0.6秒フェードイン）。別BGM再生で自動クロスフェード、同一曲なら無視 |
| `[stopbgm time=600]` | BGMフェードアウト停止 |
| `[playse storage="se_page.m4a"]` | 効果音を単発再生 |

- 既定音量: BGM 0.5 / SE 0.7（将来の設定画面で調整可能にする予定）
- 再生中BGMはオートセーブに記録され、「つづきから」で復帰
- タイトルに戻ると自動でフェード停止
- 配置先: `game/data/bgm/`（BGM）・`game/data/se/`（SE）。`tools/build_release.py` が配布ビルド時に自動でbase64インライン化する
