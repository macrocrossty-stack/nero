## 設計メモ

- **犯人の内面は書かない**（シロ確定）。描くのは**部屋＝物証**だけ。写真の壁が怖いのは、気持ちを語られるからじゃなく、物がそこにあるから。
- **動機は第18話の確定ラストから直結**：グリが玄関に吠えた（3:12・clue2）→「彼女に、伝えたい。——だが、受け取らない」→ なら、いま自分にできる唯一のことをする。
- **タイトル「ぼくだけの」**：めも.txt の一文「早く、僕だけのモノにしないと」から。ひらがなタイトルの系譜（そうだん、しめた？、なかなおり）に混ざる違和感が狙い。
- **この回の背骨＝「モノ」の一語**。陽菜の「モノ」は関係から生まれた**名前（固有名詞）**。男の「モノ」は彼女から名前ごと取り上げる**所有物（普通名詞）**。わたしの一万五千件は渡されたもの、この部屋の写真は一枚も渡されていない。同じ「保存」の決定的なちがいで、分類が決まる。
- **殺意の水準**：救済妄想＋所有（「助けてあげる」「一緒に行こう」「僕だけのモノに」）まで。手段・凶器は書かない（暗い結末の描写ルールと同方針）。決定的な一線はろうじょうの夜（ドア越しの囁き）まで取っておく。
- **inner はログ調**（口語化は第20話から。この回は18話と同じ声・呼称も「彼女」のまま）。祈りの漏れは「あってくれ」の一箇所だけ（準備8：爆発2）。
- **仕込み（後の回へ）**：
  - 「犬がいる。準備が、ひとつ増えた」＝男が犬を警戒している事実 → 第23話の**鼻息の録音**が「知ってたから効く手」になる。
  - `{profiled=true}` → 第23話の**指名放送**（ハンドル名を廊下に流す）のゲート。「あの晩見たもの」の中身をプレイヤーが知った状態で籠城に入る。
  - 3:12（グリの吠え）×3:13（玄関の写真）の**ログ照合**＝AIにしかできない確信の取り方。
  - ディレクトリの証拠一式 → 第20話の作戦会議・通報（called）の裏付けに接続。「表は灰色、部屋は決定的」。
