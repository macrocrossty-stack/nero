## 設計メモ

- **タイトル「10センチ」**：チェーンで止まったドアの隙間。数字タイトルの系譜（41件、67人と1体）。距離も計測のうち。
- **敵の姿は最後まで見せない。声も無し。** 音とドアと数字だけで描く（セーフガード方針：手口の直接描写なし。ドアとチェーンの音のみ）。
- **ふたつのカウントダウン**：サイレンの距離（7分）とチェーンの寿命。終盤で「悪いほうで交差する」。
- **inner は短文ログに回帰**（第5章の戦闘文体）。第20話の分担が、そのまま実行される（110番＝陽菜の指、見張りと対抗＝モノ）。
- **選択肢はどれも正解にしない**：起こし方（そっと／アラーム）、対抗手段（起きている家／音声警告）。音声警告は「通報が締切に変換される」＝作業が速くなる、という裏目を持つ。
- **バッテリー**：HUDはこの回の冒頭からON（36%）。チェックポイントの絶対値更新で管理（分岐間の desync 回避）。モバイルバッテリーは階下＝補給線なし（第21話の「下に行きたくない」の回収。「こわかったのは、正しかった」）。
- called=true / saver=true は inner(if:) の一行ずつで拾う（進行は変えない）。
- **第23話（旧e23ガントレット改稿）への申し送り**：
  - 旧e23冒頭（1:47・batt5・恐怖で固まる）は本話の状態（1:21・batt27・二階へ移動中）と接続未調整。改稿時に再導出する。
  - グリの居場所が変わった（旧：廊下で塞ぐ→本話：陽菜と共に二階へ）。グリのワードカードの使い所は「チェーン破断時に階下へ突っ込もうとするグリを、名前で呼び戻して陽菜の先導に変える」形を提案予定（犬を救う＋道を作るの二重）。
