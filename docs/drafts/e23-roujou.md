# 第23話「ろうじょう」初稿（未確定・レビュー待ち）

> クライマックス前半＝**見える爆弾**の回。核はひとつ：時間を稼ぐ籠城。すべての策をプレイヤーに見せて、すべて剥がされる。
> 隠し爆弾（部屋に陽菜はいない）は第24話「いないふり」で破裂する。収束は第25話へ繰り下げ。
> ※旧 e23-inaifuri.md は廃案（詰め込みすぎ）。部品はこの23話と次の24話に分けて再利用。

## 設計メモ

- **嘘は一行も書かない**。「陽菜のそばに」「済ませることをぜんぶ済ませる」「グリさんの鼻息が聞こえてるはずだ」——すべて真実。プレイヤーが「あの部屋にいる」と誤認するのは、**チャットUIに場所がないから**（テキストはどの部屋からでも打てる）。
- **縫い目はマイ・ガールの爆音の三十秒だけ**。「この三十秒で、済ませることをぜんぶ済ませる」——初読では防衛準備、再読では移動。
- **イヤホンは第21話の録音機材から**（「機材のところに、イヤホンがある」）。唐突さの解消。
- **間取りは男の捜索の音で描く**：玄関→リビング（カーテン）→キッチン→風呂場→階段→二階廊下→突き当たりが陽菜の部屋、手前が母の部屋。
- **母の部屋の前を通過する瞬間、inner は「……。」一行だけ**（言葉にならない待機。初読は不気味な間、再読は種）。
- **「サイレンは、まだ。」の反復**で時間を刻む。男の慎重さ・犬への恐怖・ノックの儀式性——彼の性質そのものを時間に変換する。
- **鼻息は録音**（第4話の「奇跡の瞬間」一万五千件アーカイブ。種明かしは24話）。グリ本体はずっと陽菜のそば＝安全。
- ⟦はんぶんこ⟧の回収（[要:]の温度選択肢）。ai(中断)「いま、そっちに」＝未送信モチーフの危機形。
- 幕切れの「いいだろう。開けてみろ。」——初読は捨て身の挑発、再読は招待。
- ゲート（c_move / c_dog / c_star / c_half）は24話の分岐材料。24話で [要:ステレオ] と corner（Gルート入口）を扱う。
- バッテリー：27→19→14→11→8。
